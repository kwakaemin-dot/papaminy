## 1. 🌍 트렌드 해킹 분석 — 조회수를 끌어당기는 7가지 패턴

수집된 떡상 영상 10개를 교차 분석한 결과, **알고리즘이 좋아하는 트리거**는 다음 7가지로 수렴합니다.

### 📊 패턴 ① **구체적 숫자 = 신뢰의 언어**
- `$62,000/Month`, `97% in 16 Minutes`, `Just ₹2 Lakh`, `2시간`, `90 DAYS`, `30일`
- 추상적 표현(`수익을 얻으세요`)보다 **정확한 수치**가 CTR을 2~3배 끌어올림
- → **액션:** 제목/썸네일에 숫자 최소 1개, 가능하면 통화 단위 포함

### 🤖 패턴 ② **"풀 오토메이션 / Clone" 신화**
- `Full Automation`, `Clone Any YouTube Channel`, `무인 자동화`
- 시청자가 원하는 건 **"내가 안 해도 된다"**는 무의식적 해방감
- → **액션:** 워크플로우의 자동화 단계를 화살표(→)로 시각화

### 💎 패턴 ③ **도구 브랜드명 = 검색 트래픽 흡수기**
- `Claude Code`, `Claude AI`, `Gemini`, `ChatGPT`가 **그 자체로 검색 키워드**
- 새 도구 업데이트 → 새 영상 사이클 = 알고리즘의 "신선도 가산점"
- → **액션:** 영상 본문에 도구명을 최소 5회 이상 노출

### 🕵️ 패턴 ④ **"숨겨진 진실 / 조용한 뉴스" 후킹**
- `Why Tech Companies Are Quietly Cancelling...`, `What China Understands That the US Doesn't`
- **"당신만 모르는 정보"** 프레임 = 클릭 충동의 정석
- → **액션:** "조용히 / 아무도 말 안 하는 / 실체" 단어를 후킹 스크립트에 배치

### 😱 패턴 ⑤ **감정 이모지 + 강한 부사**
- `😱`, `😍`, `충격에 얼어붙은`, `몰아보기`
- 한국 알고리즘은 **이모지 + 자극 부사** 조합에 약함 (유미카 케이스)
- → **액션:** 제목 후반부에 감정 단어 1개, 이모지는 1개만 (과하면 신뢰도 하락)

### 🏁 패턴 ⑥ **"챌린지 / N일 완성" 포맷**
- `90 DAYS Challenge`, `16 Minutes`, `30일 만에`
- 시청자가 **"이 영상 하나로 끝낼 수 있겠다"**는 시간적 약속을 선호
- → **액션:** "X일 / X시간 / X분" 시간 박스 구조 채택

### 🌍 패턴 ⑦ **국가·기업 비교 = 댓글 논쟁 = 알고리즘 부스터**
- `China vs US`, `India's First`, `현대자동차 vs 북한`
- 댓글창에서 찬반 토론이 일어나면 **체류시간 + 댓글 = 바이럴 트리거**
- → **액션:** "A는 이미 하고 있는데 B는 모르는" 대비 구도 설계

- 부검(Autopsy) 콘텐츠는 충성도 높은 시청자층 형성에 탁월
- → **미개척 콘텐츠:** "AI로 자동화한 채널 3개, 6개월 만에 망한 진짜 이유"
