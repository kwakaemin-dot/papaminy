- **시각적 일관성**: Visual System Guide(Black/Gold Contrast)를 모든 데이터 시각화에 반영한다.

## 3. 통합 아키텍처 제안 (Data Flow)
1.  **Source Layer (Application Logic)**: 핵심 로직에서 $S_M$ 측정 및 $S_{LER/PL}$ 계산을 수행하고, 이를 이벤트 스트림으로 발행한다.
2.  **Pipeline Layer (Real-time Stream)**: 이벤트 스트림(예: Kafka)이 $S_M$ 데이터를 수집한다.
3.  **Metric Layer (Prometheus)**: 수집된 데이터가 Prometheus 메트릭 스크랩을 통해 저장된다.
4.  **Visualization Layer (Grafana)**: Grafana를 통해 실시간으로 $S_M$과 성능 지표를 시각화하여 대시보드를 제공한다.
