코다리가 정의한 API 스키마와 LTV/CAC, 참여도 KPI를 시각화하는 핵심 대시보드 프로토타입의 초기 데이터 연동 테스트 준비를 위해, 골드와 블랙 브랜드 가이드라인을 적용한 레이아웃 및 구조 초안을 제시합니다. 이는 개발팀이 즉시 백엔드 API와 연결하여 데이터 흐름(DFRS)을 검증할 수 있는 기반이 됩니다.

## 📊 대시보드 프로토타입 설계 (골드 & 블랙 시스템 적용)

**브랜드 비주얼 적용:**
*   **메인 배경:** 딥 블랙 (`--color-pitch-black`) 또는 다크 그레이 계열로 설정하여 데이터의 대비(Contrast)를 극대화하고 고급스러운 신뢰감을 부여합니다.
*   **강조 색상 (Accent Color):** 골드 (`--color-gold`)를 핵심 KPI 값, 그래프 선, 그리고 CTA 버튼에 사용하여 부와 성과를 시각적으로 강조합니다.
*   **타이포그래피:** 제목과 주요 수치는 산세리프 계열의 강력하고 명료한 폰트를 사용하여 정보 전달의 명확성을 확보합니다.

### 1. 대시보드 레이아웃 구조 (Wireframe Concept)

| 영역 | 구성 요소 | 디자인 특징 (골드/블랙 적용) | 목적 |
| :--- | :--- | :--- | :--- |
| **A. 헤더 (Header)** | 로고 / 현재 상태 표시 | 블랙 배경, 골드 텍스트로 브랜드 아이덴티티 강조. | 브랜드 인지 및 전체 화면의 톤 설정. |
| **B. 핵심 KPI 요약 (Summary KPIs)** | LTV/CAC 비율, 참여도(Engagement Rate), 총 수익 등 주요 지표 카드 | 각 KPI는 독립된 골드 박스 내부에 배치하여 시각적 무게감을 부여. | 사용자가 한눈에 비즈니스 성과를 파악. |
| **C. 데이터 흐름 시각화 (Data Flow Visualization)** | DFRS 기반의 핵심 데이터 스트림 그래프 (예: 전환율 추이, API 호출량) | 블랙 배경 위에 골드 라인/채우기로 시각화하여 데이터의 '흐름'과 '성장'을 강조. | 시스템 안정성과 데이터 흐름 검증에 집중. |
| **D. 상세 분석 영역 (Detailed Analysis)** | 세부 데이터 테이블 및 추이 차트 (API 연동 테스트 대상) | 블랙 배경, 골드 테두리 또는 골드 색상의 라벨링으로 명확하게 구분. | API 응답 데이터를 정밀하게 검증하는 공간 제공. |
| **E. 액션 영역 (Action Area)** | 다음 단계 제안 / 시스템 안정성 상태 표시 | 골드 버튼을 사용하여 가장 중요한 행동(예: '시스템 안정성 확인')을 유도. | 사용자의 다음 행동 경로 제시. |

### 2. 핵심 KPI 시각화 세부 정의

**KPI 1: LTV/CAC 비율 (투자 효율성)**
*   **시각화 방식:** 게이지 바 (Gauge Bar) 또는 진행률 표시기 (Progress Bar).
*   **색상 코드:** 목표치(3:1 이상)에 도달하면 골드 색상이 채워지며, 미달 시 붉은색 경고선이 함께 표시됩니다.

**KPI 2: 참여도 (Engagement Rate)**
*   **시각화 방식:** 선 그래프 (Line Chart) 및 비교 막대 차트 (Bar Chart).
*   **디자인 규칙:** '긴장 $\rightarrow$ 해소 $\rightarrow$ 기대' 흐름을 반영하여, 시간 경과에 따른 참여도 변화를 부드러운 골드 선으로 표현합니다.

### 3. 데이터 연동 테스트 준비 사항

1.  **API 스키마 매핑 확인:** 코다리가 정의한 각 KPI(LTV/CAC, 참여도)에 대응하는 API 엔드포인트와 반환되는 데이터 필드가 프로토타입의 시각화 요구사항과 **1:1로 매핑**되는지 확인합니다.
2.  **데이터 흐름 검증:** DFRS에 따라 데이터가 백엔드에서 정확하게 추출되어 프론트엔드로 전달되는지, 특히 비동기 데이터 처리 부분에서 병목 현상이 없는지 테스트 준비를 완료합니다.
3.  **스타일 일관성 점검:** 모든 시각화 요소(색상, 폰트 크기)가 '골드와 블랙' 시스템을 벗어나지 않는지 최종 검토하여 디자인 가이드라인 준수 여부를 확인합니다.
