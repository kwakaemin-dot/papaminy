### 🔥 알고리즘이 좋아하는 7가지 공통 DNA

**① 숫자 트리거 (Number Trigger)**
- "10가지", "2시간", "90 DAYS", "30 Minute" — **구체적 숫자 1~2개는 무조건 박혀있음**
- 추상적 표현 ❌ → "확 줄여주는" ❌ → **"2시간 줄여주는"** ⭕
- 알고리즘은 숫자가 들어간 타이틀을 정보 밀도 높게 인식 → CTR ↑

**② 도구 이름 노출 (Tool Branding)**
- 제미나이, Claude AI — **AI 제품명이 제목에 직접 등장**
- 검색 유입의 핵심 진입로 (브랜드명 = 롱테일 검색어)

**③ 자동화 = "대신 해줌" 약속**
- "Full Automation", "30분 만에", "작업시간 줄여주는"
- **인간의 노동력을 AI가 대체한다는 판타지**가 1차 시청 동기

**④ 감정 폭격 헤드라인**
- 🚨, 😱, 😍, 충격, 얼어붙은, Dangerous — **이모지/감정어 1개 이상 필수**
- 감정적 호기심이 임펄스 클릭을 유도

**⑤ 챌린지 / 서사 구조**
- "90 DAYS Challenge" — 단순 정보가 아닌 **여정 형식**은 체류시간 ↑
- 알고리즘은 시리즈 시청을 유도하는 콘텐츠를 더 밀어줌

**⑥ 다국어 시장 동시 점화**
- 🇰🇷 🇺🇸 🇮🇳 — **AI 자동화는 글로벌 보편 주제**
- 각 언어권별 "로컬 AI 도구 + 로컬 페이먼트 수단" 조합이 떡상

**⑦ 반전/금기 어휘**
- "AI छोड़ो!" (AI 그만 써!), "Dangerous", "충격" — **기존 관습을 깨는 단어**는 호기심 GAP을 생성

### 🧠 알고리즘 핵심 공식
```
높은 CTR = (구체적 숫자) × (감정 단어) × (도구명) × (자동화 약속)
```

- **실제 손해액, 실패율 %, 왜 망했는지**를 공개하면 신뢰도 + 호기심 동시 확보
- 키워드: "AI 자동화 실패 사례", "유튜브 자동화 손해"

**🥈 2순위: "수치 검증형 AI 수익 공개"**
- "수익 NNN만원", "조회수 00만", "구독자 +00명" — **실제 스크린샷 + 대시보드** 공개
- "AI 자동화 실제 수익", "유튜브 자동화 매출"

**🥉 3순위: "비교 실험 영상 (A/B Test)"**
- "AI 도구 5개 돌려봤다, 결과는 충격" — **랭킹 + 승자 공개**
- 시청자 욕구: "어떤 도구가 진짜인지 모르니까 누가 검증해줘"
- 키워드: "AI 도구 비교", "챗GPT vs 제미나이 vs 클로드"

**4️⃣ 4순위: "0원/소자본 AI 자동화"**
- "투자금 0원, 영어 0", "초등학생도 가능" — **진입장벽 제거**
- 한국에서 압도적 검색량 보유

**5️⃣ 5순위: "AI 자동화 + 전문직 킬러 콘텐츠"**
- 번역가, 디자이너, 코더를 **AI가 대체하는 구체적 시나리오**
- "이 직업 3년 안에 사라집니다" 류의 직업 위기형

**6️⃣ 6순위: "AI 자동화의 어두운 면 / 윤리"**
- "저작권", "Deepfake", "AI가 만든 가짜뉴스" — 사회적 임팩트
- 메인 알고리즘보다 **추천 알고리즘 + 댓글 폭발**로 바이럴

**7️⃣ 7순위: "특정 국가/플랫폼 한정 자동화"**
- "한국에서만 가능한 AI 자동화", "인스타 + AI", "틱톡 + AI"
- 거대 키워드 대신 **장기 롱테일 SEO 점령**

### 🎯 추천 진입 조합
> **빈집 1순위(실패) + 2순위(수익 공개) + 3순위(도구 비교)**를 하나의 시리즈로 묶으면
> "검증된 정보 채널" 포지셔닝이 3개월 안에 확립됩니다.

## 🎬 3. 파괴적 영상 기획안

### 🏆 추천 컨셉
> **"AI로 유튜브 자동화 6개월 한 한국인의 충격적인 진실"**
> *(역설적·부정적·구체적 숫자 — 위 5가지 패턴 모두 적용)*

---

### 🖼️ 썸네일 카피 (3안)

| 안 | 카피 | 비주얼 |
|---|---|---|
| **A** | **"6개월 만에 -380만원"** | 적색 숫자 크게 + 충격받은 본인 얼굴 + AI로 가득 찬 컴퓨터 화면 |
| **B** | **"AI가 내 채널을 죽였습니다"** | 유튜브 재생 버튼 위에 금이 간 글자 + 어두운 배경 |
| **C** | **"99%가 모르는 자동화의 거짓말"** | 권총 위에 "AI" 글자 쓴 콜라주 (의미: 무기로 둔갑한 AI) |

> 🎨 **공통 원칙:** 글자 ≤ 7개, 숫자 1개 이상, 감정(충격/분노) 표정, 대비 강한 색상

---

### 📝 제목 3개

```
1. AI로 유튜브 자동화 6개월 했더니 월 -380만원 버는 중입니다 (진짜 데이터)
2. 99%가 모르는 유튜브 자동화의 거짓말 — 6개월 부검 결과
3. AI가 내 채널을 죽였습니다. 6개월간 진짜损益을 전부 공개합니다
```

**제목 공식:** `[도전 기간] + [충격 수치/반전] + [구체성 단어(진짜, 전부, 부검)]`

---

### 🎙️ 후킹 오프닝 (첫 5초 스크립트)

> **[0초 — 화면: 마이크 앞에서 화난 표정으로 노트북 닫는 장면]**
>
> *"여러분, AI로 유튜브 자동화해서 6개월 동안 번 돈이 —"*
>
> **[2초 — 화면 전환: 적색 -380,000원 자막 + 현금 이미지]**
>
> *"— 마이너스 380만원입니다."*
>
> **[3.5초 — 빠르게 줌인]**
>
> *"지금부터 이 6개월간의 진짜损益, 알고리즘이 어떻게 망가졌는지, 그리고 —"*
>
> **[5초]**
>
> *"— AI가 도대체 어디서부터 잘못된 건지, 전부 까보겠습니다."*

---

### 🔬 후킹 설계의 5가지 알고리즘 트리거

| 초 | 트리거 | 알고리즘 효과 |
|---|---|---|
| 0~2초 | **부정적 결과 (실패)** | 이탈률 ↓ (호기심) → 평균 시청 지속시간 ↑ |
| 2~3초 | **구체적 숫자 (-380만원)** | 신뢰도 ↑ → 댓글/공유 ↑ |
| 3~5초 | **"까보겠습니다" (진실 공개 약속)** | "보상 예측" 심리 작동 → 끝까지 시청 |
| 전체 | **자조 + 진정성** | 공감 댓글 폭발 → 알고리즘 재추천 |
| 5초 후 | **질문형 구조 유지** | "그래서 어떻게 됐어?" — 다음 클릭 유도 |

---

### 📈 예상 성과 시뮬레이션

| 지표 | 보수적 | 낙관적 |
|---|---|---|
| CTR | 6.5% | 11% |
| 평균 시청 지속시간 | 4분 20초 (45%) | 7분 10초 (75%) |
| 댓글 수 | 800개 | 3,000개+ |
| 알고리즘 진입 | 24시간 내 | 6시간 내 |

> ⚠️ **리스크 관리:** 실패 스토리는 시청자 이탈 후에도 **"교훈·인사이트"** 로 마무리를 못 박으면 브랜드 신뢰가 떨어집니다. **반드시 마지막 30초에 "그래도 내가 배운 3가지"** 클로징 필수.

### 🛠️ 제작 시 즉시 실행 체크리스트

- [ ] 첫 5초 안에 **숫자 + 부정부** 동시 노출
- [ ] 썸네일 표정 **과장 200%** (자연스러움 0%)
- [ ] 제목에 **기간(6개월) + 수치 + 감정 단어** 모두 포함
- [ ] 본문 1분마다 **반전 포인트** 배치
- [ ] 댓글 핀 고정: *"혹시 여러분도 이런 경험 있으신가요?"* → 알고리즘 대화 신호 폭발
- [ ] 24시간 내 업로드 후 **30분 단위 핑퐁 댓글** (초기 알고리즘 부스트)
